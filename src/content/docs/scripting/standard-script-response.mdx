The standard script response is a very common type returned by trust scripts.
It will either be an object containing one `ok` property with the value `true`.
Or it will be an object with an `ok` property with the value `false` and an optional `msg` property that contains an
error message as a string if present.

## CLI
When this type is returned to the CLI, the client automatically converts it into a `Success` or `Failure` message.
