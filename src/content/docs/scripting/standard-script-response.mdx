The standard script response is a very common type returned by trust scripts.
It will either be an object containing one `` `Nok` `` property with the value `` `Vtrue` ``.
Or it will be an object with an `` `Nok` `` property with the value `` `Vfalse` `` and an optional `` `Nmsg` `` property
that contains an error message as a string if present.

## CLI
When this type is returned to the CLI, the client automatically converts it into a `` `LSuccess` `` or `` `DFailure` ``
message.

:::caution
[`` `Cecho`.`Lecho` `` is a player script.](/scripting/echoecho)
:::

## Type
```ts
type StandardScriptResponse = { ok: true } | { ok: false, msg?: string }
```
