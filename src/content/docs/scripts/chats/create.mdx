Create a new chat channel.

## CLI
### ``` `Fchats``A.``Lcreate` `A{` `Nname``A:` `V"<string>"` `A}` ```
Create a channel without a password. The channel name is not automatically published anywhere so it will remain
somewhat private. The channel name can be up to 50 characters.

### ``` `Fchats``A.``Lcreate` `A{` `Nname``A:` `V"<string>"``A,` `Npassword``A:` `V"<string>"` `A}` ```
Create a channel with a password. This makes it so ``` `Fchats``A.``Ljoin` ``` requires a password to join the channel.

## Subscript
```js
#fs.chats.create({ name: "foo" })
#fs.chats.create({ name: "foo", password: "bar" })
```

### Args
None.

## Return
An object containing containing boards (their titles and slugs) and posts.
